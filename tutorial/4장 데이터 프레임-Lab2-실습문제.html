Data Frame¶

  • Pandas는 라벨이 붙어있는 2차원 배열(DataFrame)이나 1차원 배열(Series)을 다루는 도구
  • 2차원 배열(DataFrame)의 행에 붙어 있는 라벨을 index(인덱스), 열에 붙어 있는 라벨을 column(컬럼)이라 부름
  • 1차원 배열에는 행에만 라벨이 붙어 있고, 이를 index(인덱스)라고 부름

Lab 2 : 데이터 프레임 만들기¶

  • 이름을 index로 만들기
In [3]:
# 데이터 프레임을 처리하기 위해 pandas 패키지를 로드
import pandas as pd
In [4]:
# Data Frame 생성(컬럼 이름과 인덱스 이름을 명시적으로 지정)
df2 = pd.DataFrame([[90, 50], [80, 60], [60, 100], [70, 20]],
                    columns = ['english', 'math'],
                    index   = ['김기훈', '이유진', '박동현', '김민지'])
df2
Out[4]:
english math
김기훈 90 50
이유진 80 60
박동현 60 100
김민지 70 20
In [5]:
# df2의 index 값 출력
df2.index
Out[5]:
Index(['김기훈', '이유진', '박동현', '김민지'], dtype='object')
In [6]:
# df2의 컬럼 값 출력
df2.columns
Out[6]:
Index(['english', 'math'], dtype='object')
In [7]:
# df2의 데이터 값 출력(array)
df2.values
Out[7]:
array([[ 90,  50],
       [ 80,  60],
       [ 60, 100],
       [ 70,  20]], dtype=int64)
In [8]:
# df2의 컬럼 math의 평균 구하기
df2['math'].mean()
Out[8]:
57.5
In [9]:
# df2의 컬럼별 값이 50이상 여부 확인 (True, False)
df2 > 50
Out[9]:
english math
김기훈 True False
이유진 True True
박동현 True True
김민지 True False
In [10]:
# True : 1, False : 0
(df2 > 50).sum()
Out[10]:
english    4
math       2
dtype: int64
